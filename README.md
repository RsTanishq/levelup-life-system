# LevelUp Life System 🎮

LevelUp Life System is a gamified self-improvement platform inspired by RPG progression mechanics like Solo Leveling.

The platform transforms real-life activities such as studying, fitness, and productivity habits into character stats, XP progression, streak bonuses, achievements, guild collaboration, and unlockable rewards.

Users level up their real-world skills as if they were characters inside a game.

---

## Project Vision 🌍

Traditional productivity tools track tasks.

LevelUp Life System builds a **life progression engine**.

Users:

- gain XP from real activities
- increase character stats
- unlock achievements
- join guilds
- complete boss challenges
- earn rewards
- compete with friends
- participate in seasonal events

The system acts as a **personal RPG dashboard for real-life growth**.

## Getting Started ⚙️

Clone repository:
git clone <repository-url>


Install dependencies:

### Backend


cd backend
npm install
npm run dev


### Frontend


cd frontend
npm install
npm run dev
